Studying <HTML>

Common Reserved Characters

In HTML, the following characters are reserved and must be represented by entities:

  • Less than: <
  • Greater than: >
  • Ampersand: &
  • Double quote: "
  • Single quote: '

Example Usage

To display a less than sign, use < instead of <.

To display a greater than sign, use > instead of >.

To display an ampersand, use & instead of &.

To display double quotes, use " instead of ".

To display single quotes, use ' instead of '.

Character Entities

Character entities are used to display characters that are reserved in HTML or those that might not be readily available on a keyboard.

Named Entities

  • < for less than
  • > for greater than
  • & for ampersand
  • " for double quote
  • ' for single quote

Numeric Entities

  • < for less than
  • > for greater than
  • & for ampersand
  • " for double quote
  • ' for single quote

Example Usage with Numeric Entities

Numeric entity for less than: <

Numeric entity for greater than: >

Numeric entity for ampersand: &

Numeric entity for double quotes: "

Numeric entity for single quotes: '
